Slide 7 (Slide 4)
------------------------------------------------------------------------
Title: Validating AI Before Full Deployment

Subtitle: Steps to test before going live

Content:
• Run the AI output against a small, low-risk pilot project first
• Check results manually
• Compare AI-generated designs with proven, human-verified benchmarks
• Involve a qualified engineer or domain expert to review outputs before sign-off
• Document every discrepancy found during testing, no matter how minor
• Only scale up once the tool has passed consistent, repeatable checks

Slide 8 (Slide 5)
------------------------------------------------------------------------
Title: Key Points to Remember

Subtitle: Core principles for safer AI design

Content:
• Always validate AI-generated designs against real-world engineering standards before use
• Human review is non-negotiable
• Treat AI output as a starting point, not a finished solution
• Document every change made to an AI-generated blueprint so decisions stay traceable
• Involve domain experts early, especially when safety is a factor
• As a next step, audit one recent AI-assisted project using a structured validation checklist
